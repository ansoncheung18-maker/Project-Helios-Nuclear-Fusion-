## 1. 退役條件

喺以下任何一種情況下，任務將進入退役程序：

| # | 條件 | 說明 |
|:---|:---|:---|
| 1 | **設計壽命屆滿** | 營運滿 60 年 |
| 2 | **災難性故障** | 反應堆核心無法修復 |
| 3 | **成本效益過低** | 年營運成本 > 年收入 |
| 4 | **技術過時** | 新技術取代（如純聚變成功） |
| 5 | **燃料供應中斷** | 氘燃料無法補充 |
| 6 | **監管要求** | 核能法規變更 |

## 2. 關閉程序

### 2.1 關閉順序

| 步驟 | 動作 | 說明 |
|:---|:---|:---|
| 1 | 停止 D-D 反應 | 降低等離子體參數，自然熄火 |
| 2 | 停止發電 | 斷開電網連接 |
| 3 | 冷卻反應堆 | 繼續冷卻至 100°C 以下 |
| 4 | 卸載所有小爐 | 機械臂將 870 個小爐移至冷卻池 |
| 5 | 排空冷卻劑 | PbLi 冷卻劑回收 |
| 6 | 卸載超導磁體 | 消磁後移除 |
| 7 | 關閉控制系統 | 最後數據備份 |
| 8 | 圍阻體封閉 | 混凝土密封入口 |

### 2.2 預計關閉時間

| 階段 | 時間 | 備註 |
|:---|:---|:---|
| 準備工作 | 1 個月 | 通知、計劃 |
| 反應堆熄火 | 1 週 | 逐步降功率 |
| 小爐卸載 | 2 個月 | 870 個小爐 |
| 設備拆除 | 3 個月 | 磁體、冷卻系統 |
| 圍阻體封閉 | 1 個月 | 混凝土密封 |
| **總關閉時間** | **約 7 個月** | |

## 5. 設備處置

### 5.1 小爐處置

| 選項 | 說明 | 可行性 |
|:---|:---|:---|
| 1. 永久儲存 | 存放於核廢料儲存設施 | ✅ 最可行 |
| 2. 回收材料 | 回收貧鈾/釷、不鏽鋼 | ⚠️ 成本較高 |
| 3. 深地質掩埋 | 埋入深層地質層 | ✅ 可行 |

### 5.2 其他設備處置

| 設備 | 處置方式 | 備註 |
|:---|:---|:---|
| 超導磁體 | 回收超導材料 | 高價值 |
| 冷卻劑 (PbLi) | 回收再利用 | 可重複使用 |
| 反應堆容器 | 封閉於原地 | 混凝土密封 |
| 渦輪發電機 | 拆解回收 | 金屬回收 |

### 5.3 推薦方案

**小爐：永久儲存於核廢料設施**
**反應堆容器：封閉於原地，作為「人類首座核聚變電廠」紀念碑**

| 優點 | 缺點 |
|:---|:---|
| 安全、可行 | 長期監管需要 |
| 具歷史意義 | - |
| 無需額外運輸 | - |
