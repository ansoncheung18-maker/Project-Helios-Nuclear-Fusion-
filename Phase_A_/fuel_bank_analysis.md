---

## 1. 問題：月球氦-3開採成本過高

根據月球氦-3開採模擬結果：

| 參數 | 數值 |
|:---|:---|
| 年需氦-3量 | 350 kg |
| 需處理月壤 | 5,000 萬噸/年 |
| 需要挖掘機 | 63 台 |
| 年開採成本 | $3.50 億美元 |
| 電廠年利潤 (不含開採) | $1.67 億美元 |
| **電廠年利潤 (含開採)** | **-$1.83 億美元 (虧損)** |

❌ 月球開採方案經濟上不可行

---

## 2. 解決方案：燃料銀行 (Fuel Bank)

### 2.1 原理 (Helion Energy 概念)

採用氘-氘 (D-D) 反應作為「種子」，在反應爐內自然產生氦-3：

| 反應 | 產物 | 說明 |
|:---|:---|:---|
| D + D → T + p | 氚 + 質子 | 副反應 |
| D + D → He-3 + n | **氦-3 + 中子** | **主反應** |
| D + He-3 → He-4 + p | 氦-4 + 質子 | 能量反應 |

**關鍵優勢**：氦-3 在爐內循環使用，無需外部補充。

### 2.2 燃料銀行 vs 月球開採

| 對比項 | 月球開採 | 燃料銀行 |
|:---|:---|:---|
| 年成本 | $3.50 億 | **$0.5 億** |
| 技術成熟度 | 低 (需 10+ 年) | 中 (Helion 已有原型) |
| 依賴月球 | 是 | 否 |
| 規模化難度 | 高 | 低 |
| 年利潤 | **-$1.83 億** | **+$1.17 億** |

✅ 燃料銀行方案經濟可行

| 參數 | 數值 |
