# 核聚變專案 Phase A：概念設計與可行性論證

**專案名稱：太陽神計劃 (Project Helios)**  
**版本：1.1**  
**更新日期：2026-06-02**  
**核心目標：建造首座「模塊化聚變-裂變混合電廠」，實現清潔、穩定、可盈利嘅核聚變能源。**

---

## 1. 願景與目標

> 如果核聚變成功，人類文明將進化到全新階段。本專案旨在將核聚變從「科學問題」轉變為「工程問題」，並提供一條清晰、可行、經濟誘人嘅實現路徑。

### 1.1 終極目標

- 建造一座 500MW 模塊化聚變-裂變混合電廠
- 淨輸出 500 MWe，年發電約 4,000 GWh
- 實現 **約 4 年** 投資回收期（遠低於任何現有能源基建）
- 採用 **燃料銀行** 方案，無需依賴月球開採

### 1.2 核心創新

| 創新點 | 說明 | 解決嘅問題 |
|:---|:---|:---|
| **模塊化小爐陣列** | 數百個 1m³ 可更換式小爐，取代單一巨型反應堆 | 繞過材料科學瓶頸 |
| **聚變-裂變混合** | 利用裂變倍增能量，降低對純聚變 Q 值嘅要求 | 科學門檻從 Q>10 降至 Q>1 |
| **快速更換設計** | 機械臂熱插拔，故障小爐即時更換，不停機 | 可用率從 50% 提升至 99.4% |
| **AI 實時控制** | 毫秒級監測 + 故障預測 + 自動修正 | 解決等離子體不穩定性 |
| **燃料銀行** | 用氘-氘 (D-D) 反應自產氦-3 | 解決燃料來源問題，無需月球開採 |

## 2. 技術架構

### 2.1 總體結構
初始氘燃料注入
↓
燃料銀行 (D-D 反應自產 He-3)
↓
模塊化小爐陣列 (700 個同時燃燒)
↓
巨型反應堆容器 (容納小爐、超導磁體)
↓
冷卻系統 (PbLi + 超臨界 CO₂)
↓
熱交換器 → 渦輪發電機 → 電網

### 2.2 反應堆容器

| 參數 | 數值 | 說明 |
|:---|:---|:---|
| 類型 | 地下式鋼結構環狀隧道 | 地震隔離、輻射屏蔽 |
| 直徑 | 20 米 | 容納小爐陣列 + 超導磁體 |
| 深度 | 600-800 米 | 利用地層隔熱、減少地面風險 |
| 超導磁體 | 高溫超導 (HTS) | 產生 5-10T 磁場約束等離子體 |

### 2.3 模塊化小爐 (核心創新)

| 參數 | 數值 | 說明 |
|:---|:---|:---|
| 尺寸 | 1m x 1m x 1m | 標準化、可流水線生產 |
| 數量 | 700 個 | 淨輸出 500 MWe |
| 結構 | 5 層一次性壁材 | 不鏽鋼外殼 + ODS 第一壁 + 燃料層 + 裂變包層 + 屏蔽層 |
| 燃料 | 初期氘 (D)，後期自產 He-3 | 燃料銀行概念 |
| 裂變材料 | 貧鈾 / 釷 | 聚變中子引發裂變，能量倍增 |
| 壽命 | 約 1 年 | 機械臂更換，年更換一次 |

### 2.4 燃料供應：燃料銀行 (Fuel Bank)

本專案採用 Helion Energy 燃料銀行概念，用氘-氘 (D-D) 反應自產氦-3：

| 反應 | 產物 | 說明 |
|:---|:---|:---|
| D + D → He-3 + n | **氦-3 + 中子** | 主反應，產生燃料 |
| D + He-3 → He-4 + p | 氦-4 + 質子 | 能量反應 |

**關鍵優勢**：
- 初始只需氘燃料（從海水提取，地球無限）
- 氦-3 在爐內循環使用，無需外部補充
- 年燃料成本僅 **$0.5 億**，遠低於月球開採 ($3.5 億)

| 對比項 | 月球開採 | 燃料銀行 |
|:---|:---|:---|
| 年成本 | $3.50 億 | **$0.5 億** |
| 技術成熟度 | 低 (需 10+ 年) | 中 (Helion 已有原型) |
| 依賴月球 | 是 | 否 |
| 年利潤 | -$1.83 億 | **+$1.27 億** |

✅ 燃料銀行方案使電廠從虧損轉為盈利
